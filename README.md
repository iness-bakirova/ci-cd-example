# Construction Company Management System

Это веб-приложение на Java (Spring Boot), предназначенное для управления строительной фирмой, проектами, сотрудниками и ресурсами.

## Модели и структура данных

1. **User** – базовый класс для всех пользователей.  
   Поля: ФИО, должность, контактные данные.  

2. **ConstructionWorker** – рабочие, наследуются от User.  
   Поля: специальность, ставка, квалификация.  
   Методы: вывод профиля, расчет стоимости рабочего времени.  

3. **Engineer** – инженеры, наследуются от User.  
   Поля: уровень, список проектов.  
   Методы: оценка проектов.  

4. **ConstructionSite** – строительный проект.  
   Поля: название, заказчик, стоимость, сроки.  
   Ассоциации: список инженеров и рабочих.  

5. **Company** – строительная фирма.  
   Поля: список сотрудников и проектов.  
   Методы: добавление/удаление проектов, расчет общей загрузки.

## Используемые технологии

- Java 17
- Spring Boot
- Spring Data JPA
- Spring Security
- Hibernate
- Thymeleaf
- Maven

## Возможности приложения

- Регистрация и аутентификация пользователей
- Управление сотрудниками (инженеры и рабочие)
- Создание и управление строительными проектами
- Расчет стоимости рабочего времени
- Управление материалами и ресурсами
- Просмотр отчетов по проектам и сотрудникам
