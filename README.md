# АРМ администратора автосалона

## Описание проекта

АРМ администратора автосалона — автоматизированное рабочее место для управления основными бизнес-процессами автосалона. Система предназначена для администраторов и позволяет централизованно управлять автомобилями, клиентами, продажами, сотрудниками и отчетами.

Проект направлен на повышение эффективности работы, снижение количества ручных операций и минимизацию ошибок учета.

---

## Цель проекта

Создание программного средства для автоматизации деятельности администратора автосалона.

### Задачи проекта

* ведение каталога автомобилей;
* управление клиентской базой;
* учет продаж и сделок;
* администрирование сотрудников;
* формирование отчетности;
* контроль статусов автомобилей.

---

## Роли пользователей

### Администратор автосалона

Имеет полный доступ к системе и может:

* добавлять, редактировать и удалять данные;
* управлять пользователями и ролями;
* просматривать отчеты и аналитику;
* контролировать корректность информации.

---

## Функциональные возможности

### Управление автомобилями

* добавление автомобилей;
* редактирование характеристик (марка, модель, год, цена);
* изменение статуса (в наличии, продан, забронирован);
* поиск и фильтрация.

### Управление клиентами

* добавление и редактирование клиентов;
* хранение контактной информации;
* просмотр истории покупок.

### Управление продажами

* оформление сделок;
* привязка клиента к автомобилю;
* учет стоимости и даты продажи;
* автоматическое обновление статуса автомобиля.

### Управление сотрудниками

* добавление сотрудников;
* назначение ролей;
* редактирование учетных данных.

### Отчетность

* отчеты по продажам;
* отчеты по выручке;
* статистика за выбранный период.

## Архитектура приложения

Приложение реализовано на основе многоуровневой архитектуры:

* уровень представления (UI);

* уровень доступа к данным.

Архитектура обеспечивает модульность, масштабируемость и удобство сопровождения.
